# AeroStream-Analytics

AeroStream-Analytics: pipelines and APIs for forecasting and sentiment analysis.

Quick start

- Create a Python environment (recommended: venv/conda) and install requirements from `requirements/api.txt` or other requirements files in `requirements/`.

- Run the FastAPI apps (examples):

  - API prediction service: `api/fastapi_predict.py`
  - API tweet service: `api/fastapi_tweet.py`

Development

- DAGS and Airflow related code: `airflow/`
- Notebooks: `notebooks/`
- Models: `models/`

Git

- I commit locally; run `git push origin main` to publish.
